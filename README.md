## Getting Started
you can view the deploy in github page following link: https://bryanngutierrez.github.io/cv/ or clone repository and run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
